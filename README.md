# Introduction 
Repo _temporaire_ pour le Tribunal Administratif du Logement (TAL).

Le bail 53000 est le bail commun que la majorité des personnes rempliront.
Les autre baux doivent prendre exemple sur ce dernier et lui ressembler, outre leur section spécifiques différentes et les termes différents.

Nous ne pouvons pas inventer des termes ou formuler les phrases de la manière qu'on veut. Il faut au maximum utiliser les phrases provenant des pdf de binding, ou alors les phrases doivent être approuvées par le TAL.

Ajouter les annotations (infobulles)/ avis formulés par le TAL dans les formulaires.

Chaque bail pouvant être associé à plus de deux locataires, et donc les locataires supplémentaires ne pourraient pas être bind dans le pdf de bail original, doivent avoir autant d'annexe locataire produites qu'il y a de locataires supplémentaires.
